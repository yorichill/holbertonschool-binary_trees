This project focuses on implementing various functions to manipulate **binary trees** in the C programming language.  
It covers fundamental concepts such as:

- What a binary tree is
- The difference between a Binary Tree and a Binary Search Tree (BST)
- The possible time complexity improvements compared to linked lists
- Definitions of **depth**, **height**, and **size** of a binary tree
- Different traversal methods: *preorder*, *inorder*, *postorder*, *level-order*
- Definitions of **complete**, **full**, **perfect**, and **balanced** binary trees
